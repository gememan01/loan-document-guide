대출 필요서류 안내 자동화

도움말

기본 사용 순서

  • 먼저 대출 구분에서 가계대출 또는 기업대출을 선택합니다.
  • 큰 틀에서 개인, 법인, 추가 조건 중 해당 업무 범위를 먼저 선택합니다.
  • 공통 소득서류에서 채무자의 소득 유형을 선택합니다.
  • 가계대출은 필요 시 배우자 소득서류를, 기업대출은 필요 시 기장사업자 여부를 선택합니다.
  • 오른쪽 세부 항목을 체크하면 출력 화면에 필요서류가 표시됩니다.

연도·부수 조정

소득금액증명원, 원천징수영수증, 표준재무제표처럼 연도 선택이 필요한 서류는 연도 지정에서 선택합니다. 초본, 인감증명서처럼 통수 선택이 필요한 서류는 부수 지정에서 조정합니다.

출력 화면 조정

출력 화면에서 특정 서류 옆의 삭제 버튼을 누르면 해당 안내문에서만 제외됩니다. 기본 선택을 다시 정리하려면 선택 초기화를 누르십시오.

수기 필요서류

목록에 없는 서류는 수기 필요서류에서 행 추가 후 서류명, 발급·준비 방법, 확인사항을 입력하면 출력 화면에 별도 항목으로 추가됩니다.

파일 저장/불러오기

저장명(채무자명)을 입력하고 파일 저장을 누르면 별도 저장파일이 생성됩니다. 파일 불러오기는 그 저장파일을 선택해 현재 상태로 복원합니다.

PDF 저장

PDF 저장은 인쇄창을 연 뒤 프린터 또는 대상에서 PDF 저장을 선택하면 됩니다. 휴대폰에서는 인쇄 또는 공유 화면에서 PDF 저장을 선택할 수 있습니다.

휴대폰 사용

GitHub Pages 같은 HTTPS 주소로 접속한 뒤 브라우저 메뉴에서 홈 화면에 추가하면 앱처럼 실행할 수 있습니다. 저장파일과 PDF는 휴대폰 파일 앱에 저장됩니다.

유의사항

본 화면은 안내문 작성을 돕기 위한 보조 도구입니다. 실제 심사 조건, 상품 종류, 채무자 상황에 따라 추가 서류가 필요할 수 있으므로 최종 안내 전 반드시 직접 확인하십시오.
